# VIA dataset copy

This directory contains the 342 images used by the experiments: 175 `clear.*` images and 167 `nonclear.*` images.

Canonical dataset repository: <https://github.com/fbreve/via-dataset>

If you use these images, cite:

F. A. Breve and C. N. Fischer, "Visually impaired aid using convolutional neural networks, transfer learning, and particle competition and cooperation," in *Proc. International Joint Conference on Neural Networks (IJCNN)*, 2020.

The file `data/dataset_manifest.csv` records the expected filename, class, file size, and SHA-256 hash of every image included in this artifact.
